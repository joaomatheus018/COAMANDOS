# 🚀 Guia de sobrevivencia: Dominio do Terminal (CLI)

 Este repósitorio foi criado como parte da disciplina da **Sistemas Operacionais**, ultilizando o simulador [Terminal Master](https://andersonrmgomes.github.io/terminal-master/) para praticar comandos essenciais de windows e linux.

 ---

 ## 📖 Introdução
 A interface de linha de comando (CLI) é uma ferramenta poderosa que permite interagir com o núcleo do sistema oparicional. Dominar esses comandos é fundamental para automoção de tarefas, administração de servidores e desenvolvimeto de software.

 --

 ## 🖥️ Comandos apreendidos

 Abaixo, apresento a lista dos principais comandos praticados durante o treinamento gamificado:

 |Comando |Função |Sistemas | Exemplo de uso |
 | :--- | :--- | :--- | :--- |
 | `pwd` | `Exibe o diretório atual (caminho)` | `Linux` | `pwd` |
 | `cd` |`Altera o diretorio de trabalho` | `ambos` | `cd /home/usuario/documento` |
 | `ls` | `Lista arquivos e diretórios` | `Linux` | `ls -la` |
 | `dir` | `Lista arquivos e diretórios` | `ambos` | `dir` |
 | `mkdir` | `Cria um novo diretório (pasta)` | `ambos` | `mkdir nova_pasta` |
 | `touch` | `Cria um arquivo vazio ou atualiza data` | `Linux` |`touch arquivo.txt` |
 | `type` | `Exibe o conteúdo de um arquivo de texto` | `Windows` | `type notas.txt` |
 | `rm` | `Remove (deleta) arquivos ou diretórios` | `Linux` | `rm arquivo.txt` |
 | `cp` | `Copia arquivos ou diretórios` | `ambos` | `cp foto.jpg backup/` |
 | `mv` | `Move ou renomeia arquivos/diretórios` | `ambos` | `mv antigo.txt novo.txt` |
